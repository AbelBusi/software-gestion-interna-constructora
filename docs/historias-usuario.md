# Historias de Usuario

## HU-01 – Registro de usuarios
Como administrador  
Quiero registrar nuevos usuarios  
Para que puedan acceder al sistema de gestión.

**Criterios de aceptación**
- El sistema permite ingresar nombre, correo y rol
- El usuario queda registrado correctamente

---

## HU-02 – Inicio de sesión
Como usuario  
Quiero iniciar sesión en el sistema  
Para acceder a las funcionalidades disponibles.

**Criterios de aceptación**
- Validación de credenciales
- Mensaje de error si los datos son incorrectos
